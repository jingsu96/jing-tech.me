## 前言

Lodash 在前端開發中是一個被經常被使用的 JavaScript 函式庫，其提供了很多實用的函式，讓我們可以快速對各種型別的資料進行操作。

本系列主題會透過實作 Lodash 裡常見的函式，來一起理解這些函式的底層實作原理。

每篇文章會先介紹函式的預期的功能，並會提供一個練習區，讓讀者可以嘗試自己實作，最後會附上筆者解答。

如果有任何錯誤或是更好的解法，也歡迎每篇文章的留言區提出！
